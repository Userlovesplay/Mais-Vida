## 🚀 Como Executar Localmente

### 🔧 Pré-requisitos
- Node.js 18 ou superior  
- npm ou yarn instalado  

### 💡 Passos para rodar o projeto

```bash
# Clonar o repositório
git clone https://github.com/Userlovesplay/Mais-Vida.git

# Entrar na pasta
cd Mais-Vida

# Instalar dependências
npm install

# Rodar o projeto localmente
npm run dev
```

Após isso, o projeto estará disponível em:
```
http://localhost:5173/
```

## ☁️ Deploy na Vercel

O projeto foi configurado para **deploy automático** na [Vercel](https://vercel.com/).

**Configurações recomendadas:**
- **Build Command:** `npm run build`  
- **Output Directory:** `dist`  
- **Framework Preset:** `Vite`  

Acesse o projeto online: 🌿 [mais-vida.vercel.app](https://mais-vida.vercel.app/)
